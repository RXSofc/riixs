# EvilVoid

![EvilVoid Banner](https://media.giphy.com/media/xTiTnxpQ3ghPiB2Hp6/giphy.gif)

## Overview
EvilVoid is a personal web project crafted by *Arx Riiqs* — an exploration of code, electricity, and controlled chaos.  
This site blends minimalist aesthetics, neon visuals, and smooth motion to represent the harmony between logic and creativity.

The project serves as a portfolio, a playground, and a statement that every circuit and every line of code can carry art within it.

---

## Features
- Neon cyber aesthetic built with pure HTML, CSS, and JavaScript  
- Smooth scroll and parallax hero section  
- Interactive “Message” button with a dynamic prank behavior  
- Looping background music for immersive experience  
- Responsive design optimized for both desktop and mobile
